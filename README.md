# ⌚ Tissot Clone Website

A responsive e-commerce website inspired by the Tissot brand, showcasing watches, collections, and modern UI design.

## 🚀 Live Demo
🔗 https://tissotclone.netlify.app/

---

## 📌 About The Project
This project is a frontend clone of a watch e-commerce website. It focuses on building a clean UI, structured product pages, and responsive design for a better user experience.

---

## ✨ Features
- 🛍️ Product showcase (watches & collections)
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- 🧭 Multiple pages (Home, Collections, Men, etc.)
- ⚡ Smooth navigation
